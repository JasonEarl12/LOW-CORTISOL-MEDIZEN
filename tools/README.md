# Tools Hub

This folder contains helper scripts used for debugging, maintenance, and database repair.

## Structure

- `tools/database/` - database inspection and repair scripts
- `tools/testing/` - test scripts and validation helpers

## Current scripts

Examples of scripts in this area include:
- inventory and billing repair helpers
- patient table inspection helpers
- test data insert scripts
- zero-ID cleanup scripts

## Notes

These scripts are separate from the live PHP pages so they do not change how the website works for users.
