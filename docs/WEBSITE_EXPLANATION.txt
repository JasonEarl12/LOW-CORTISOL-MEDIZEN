WHAT IS MEDIZEN?
================================================================================

MEDIZEN is a Hospital Management System - basically, it's a website that helps 
hospitals organize and manage everything related to patient care, doctors, 
appointments, and records.

Think of it as a digital office for hospitals where:
  • Patients can book appointments
  • Doctors can manage their schedules
  • Staff can track patient records
  • Administrators can manage everything
  • Hospital can keep inventory of supplies

WHY DO WE NEED THIS?
================================================================================

BEFORE MEDIZEN: Hospitals used physical files, phone calls, and manual scheduling
  • Lost or misplaced patient records
  • Long waiting times on phone
  • Booking conflicts and double bookings
  • Difficult to find patient information quickly
  • Manual billing and tracking

WITH MEDIZEN: Everything is digital and organized
  • Quick access to patient information
  • Instant appointment booking
  • Automatic appointment reminders
  • Easy billing and invoicing
  • Real-time updates for everyone

TECHNICAL SIMPLE VERSION
================================================================================

The Website Has 3 Parts:

1. FRONTEND (What You See)
   - HTML + CSS = Structure & colors
   - JavaScript = Interactive features (buttons, animations, real-time 
     updates)
   - Runs in your browser (Chrome, Firefox, etc.)
   - Everything you interact with is here

2. BACKEND (The Brain)
   - PHP = Server-side logic (processes requests)
   - Handles login, saving data, retrieving data
   - Sends data to frontend when requested

3. DATABASE (The Storage)
   - MySQL = Stores all information
   - Patient names, appointments, billing data
   - Like a giant electronic filing cabinet
   - Backend reads/writes to database

How they communicate:
YOU click button on Website
         ↓
JavaScript sends request
         ↓
PHP receives and processes
         ↓
PHP asks MySQL for data
         ↓
MySQL returns data
         ↓
PHP sends back to Website
         ↓
Website shows you the result
