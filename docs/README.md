# Documentation Hub

This folder groups the project documentation into simple categories so it is easier to find what you need without touching the website runtime files.

## Structure

- `docs/architecture/` - how the system works, folder explanations, and request flow
- `docs/reference/` - project reference material such as overviews and file explanations
- `docs/guides/` - setup, maintenance, and step-by-step instructions
